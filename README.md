```
 Version: 0.0.0-0

 A GitHub bot to automatically delete branches on your repos when PRs are merged.

  -interval int
    	Check interval in seconds (default 30)
  -repos string
    	GitHub Repos (e.g dantoml/branchbot,cocoapods/cocoapods)
  -self-only
    	Only delete branches owned by self (default true)
  -token string
    	GitHub API Token
  -verbose
    	Enable debug logging
  -version
    	Print version and exit
```
